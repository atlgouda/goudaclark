Parks List

Inspiration

I always thought the ideal road trip would be packing up an RV and touring the US national parks. I chose to create an app to learn more about each park and create a "Wish List" of which locations to visit.

Skills

This was my final project in my Web Development Immersive course, and I used the opportunity to utilize Ruby on Rails. It was a great chance to implement a new programming language and server-side web application framework. I also chose to take a deeper dive with the styled components in React. To toggle between different versions of my site, I created multiple branches in my GitHub repository.

Overview

Once a user comes to the site, they will get to a landing page to enter the site. After entering, they have the option to see the list of parks, add a park to their wish list, search parks by state, and see more of a park's information on a "show one" page.

Technologies Used

  • React
  • Ruby On Rails
  • Heroku
  • Postman
  • HTML5
  • CSS
  • GitHub
  • National Parks Service API
  • HTTParty

Guide to US national parks

Old School Jeopardy

Inspiration

Some of my favorite viral clips from Jeopardy are when Alex Trebek reads the clues from categories about rap music. I used this as an inspiration to make my Old School Rap game.

Skills

This was the first project in my Web Development course. Two weeks into the course, I built a Jeopardy based game to demonstrate my ability to use HTML and CSS to build and style an intuitive web application. I used JavaScript to show my compentancy in DOM manipulation. I began the project by organizing my tasks with Trello, and wireframing out my site. Throughout the project, I had to frequently commit on GitHub to implement version control.

Overview

Once users come to the site the game board will load. The player then can choose from 9 different questions over 3 categories. The player will select a square, and then a multiple choice question will display. As they answer questions correctly, the corresponding point total will be added to their scoreboard. The game ends when either the player reaches 100 points, or there are no more tiles to select. If a player would like to reset the game at any point, they can click the reset button.

Technologies Used

  • HTML5
  • CSS
  • JavaScript
  • jQuery
  • CSS Grid
  • Sweet Alert
  • Bootstrap
  • Google Fonts
  • GitHub
  • Netlify

Jeopardy based game for classic hip hop trivia

League of Your Imagination

Inspiration

My friends and I often have a debate about which athletes would do well in other sports on a professional level. Could Lebron be a soccer striker? What if Tom Glavine chose his NHL contract over becoming a starting pitcher. With this thought in mind I built a site in which users can create teams filled with whichever players they choose, and have them play games at any location.

Skills

This project was my full stack application. I built a Node/Express app with Mongo/Mongoose with three full CRUD models.

Overview

The user will enter into a league with some teams that have already been pre-seeded. From there they will have the ability to create/edit/delete teams, schedules, players and games. They will also be able to load images for logos, locations, and players (plus be able to create a basketball card for each player!)

Technologies Used

  • HTML5
  • CSS
  • Express
  • Bulma
  • Google Fonts
  • MongoDB
  • Mongoose
  • Node
  • Heroku
  • GitHub

An app to make the ultimate fantasy team

Restaurant Guide

Inspiration

I always try to keep a running mental list of new restauraunts I want to try, so I decided to make a tracker to keep a list of new restaurants and the neighborhoods where they are located.

Skills

This was my first project using React as a front end framework to build a mobile-first full CRUD app.

Overview

Once a user comes to site, they will get to a landing page with the title and an entry button. From there, they will come to a list of users who have already had their data loaded. From here they have the option to create a new user or view a user page that displays their selected restaurants. They can then see the restaurant page and follow a link to the restaurant's website. A restaurant can be created, updated or deleted.

Technologies Used

  • HTML5
  • CSS
  • React
  • MongoDB
  • Postman
  • Node
  • Heroku
  • GitHub

Full CRUD app for adding a user's new restaurants
